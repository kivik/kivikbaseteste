kivikbaseteste
==============

O que é kivikbaseteste?

kivikbaseteste é uma ferramenta que gera dados randômicos em massa para uma base de dados qualquer do MySQL.

Como funciona?

01. Usuário deve inserir nome da tabela.
02. Ferramenta gera apenas uma tabela.
03. Usuário insere número de colunas, número inteiro.
04. Usuário digita número de colunas.
05. Usuário insere nome das coluna e deve digitar uma letra no proximo campo, sugerido letra "v".
06. Código da criação da tabela é gerado.
07. Em desenvolvimento.
08. Dados serão gerados todos randomicamente, inclusive primary keys.
